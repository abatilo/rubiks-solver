Requires Java 8 and Gradle 4.4 to be installed.

### Build Instructions
```
git clone https://github.com/abatilo/rubiks-solver.git
cd rubiks-solver
./gradlew :solver:clean :solver:shadowJar
java -jar solver/rubiks-solver.jar
```
